## ğŸ’¡ Pro Tips

### Getting Better AI Results

1. **Photo Quality:**
   - Use natural lighting
   - Capture entire room
   - Avoid extreme angles
   - Higher resolution = better detection

2. **Room Preparation:**
   - Clear clutter
   - Show existing furniture
   - Include windows/doors for context

3. **Style Selection:**
   - Choose style that matches existing decor
   - Experiment with different styles
   - Save multiple versions

### 3D Design Best Practices

1. **Start with Essentials:**
   - Add large furniture first (sofa, bed, table)
   - Then add accent pieces
   - Finally add decor items

2. **Maintain Balance:**
   - Don't overcrowd the space
   - Leave walking paths
   - Consider real-world dimensions

3. **Use the Grid:**
   - Toggle grid for alignment
   - Helps with symmetry
   - Ensures proper spacing

4. **Save Often:**
   - Click "Save Design" regularly
   - Create multiple versions
   - Compare different layouts

### Shopping Smart

1. **Plan Before Buying:**
   - Design first, shop later
   - Check measurements
   - Verify stock availability

2. **Book from Design:**
   - Use "Book Furniture" button
   - Automatically adds design items
   - Saves time shopping

3. **Track Your Orders:**
   - Check order status regularly
   - Save order numbers
   - Contact support if needed

## ğŸ”§ Troubleshooting

### Issue: AI Analysis Taking Too Long

**Solution:**
- First analysis takes 5-10 seconds (model loading)
- Subsequent analyses are cached (instant)
- Check internet connection
- Try smaller image file

### Issue: Can't Move Furniture

**Solution:**
- Click furniture to select it first
- Look for purple highlight
- Use left mouse button to drag
- Disable orbit controls temporarily

### Issue: Cart Not Updating

**Solution:**
- Refresh the page
- Check if logged in
- Clear browser cache
- Try different browser

### Issue: Order Not Showing

**Solution:**
- Check "My Orders" page
- Verify order confirmation
- Look for flash message
- Check email (if configured)
