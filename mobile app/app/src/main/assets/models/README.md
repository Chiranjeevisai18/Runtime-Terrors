# Placeholder Models Directory

Place GLB furniture models here. For MVP, the app expects these files:

- `sofa.glb`
- `table.glb`
- `chair.glb`
- `lamp.glb`
- `bookshelf.glb`

## Sources for free low-poly GLB models:
- [Sketchfab](https://sketchfab.com/search?type=models&q=furniture&sort_by=-likeCount)
- [Poly Haven](https://polyhaven.com/)
- [Kenney Assets](https://kenney.nl/assets/category:3D)

Download low-poly furniture models in GLB format and place them in this directory.
